How to XML Schema Documenter
============================

Prerequisites
-------------

1. Microsoft Visual Studio 2010 (Professional, Premium or Ultimate).
   http://msdn.microsoft.com/en-us/vstudio/default.aspx

2. MSBuild Community Tasks
   http://msbuildtasks.tigris.org/

3. Sandcastle
   http://www.codeplex.com/Sandcastle

4. Sandcastle Style Patch
   http://www.codeplex.com/SandcastleStyles

5. Sandcastle Help File Builder (SHFB)
   http://www.codeplex.com/SHFB

6. Windows Installer XML (WiX) toolset Version 3.5
   http://wix.sourceforge.net/

Building
--------

1. Run Build\Build.bat

2. The folder Output will now contain the following subfolders:
   - Bin. Contains the binaries (compiled in "Release" configuration).
   - Help. Contains the  help file.
   - Release. Contains the zipped setup, samples, and source code.
   - Setup. Contains the setup.   
   - Samples. Contains several sample .chm files.
   - Source. Contains the complete source code.
